# File Descriptions

## .gitignore
This file specifies intentionally untracked files that Git should ignore. It includes files generated by Gradle, local configuration files, logs, Android Studio generated files, IntelliJ files, keystore files, Google Services files, and Android Profiling files.

## AGENTS.md
This file provides guidance for AI agents working on the Geministrator project. It includes an overview of the project, the architecture, the council of agents, the `AbstractCommand` and `ExecutionAdapter`, development conventions, and instructions for running tests and managing dependencies.

## LICENSE
This file contains the license for the project. The project is released into the public domain under the Unlicense.

## README.md
This file provides an overview of the MadeMeDance application. It explains how the application works, how to build it, and includes a to-do list for future development.

## ShazamKit-Android-doc-2.1.1.zip
This is a zip file containing the documentation for the ShazamKit Android SDK.

## build.gradle.kts
This is the top-level build file for the project. It configures the plugins that are common to all sub-projects/modules.

## gradle.properties
This file contains project-wide Gradle settings.

## gradlew
This is a shell script for executing Gradle tasks on Unix-like systems.

## gradlew.bat
This is a batch script for executing Gradle tasks on Windows.

## settings.gradle.kts
This file defines the project structure and includes the sub-projects/modules that are part of the project.

## app/.gitignore
This file specifies files and directories within the `app` module that should be ignored by Git. In this case, it's the `build` directory.

## app/build.gradle.kts
This is the build file for the `app` module. It applies Android and Kotlin plugins, configures the Android application settings (like `namespace`, `compileSdk`, `defaultConfig`, `buildTypes`, etc.), and declares the dependencies for the application.

## app/proguard-rules.pro
This file contains ProGuard rules for the application. ProGuard is used to shrink, obfuscate, and optimize the code.

## app/src/androidTest/java/com/hereliesaz/mademedance/ExampleInstrumentedTest.kt
This file contains an instrumented test that runs on an Android device. It checks that the app's context is correct.

## app/src/main/AndroidManifest.xml
This is the main manifest file for the Android application. It declares the application's components (like the main activity), permissions, and other essential information.

## app/src/main/java/com/hereliesaz/mademedance/AudioBpmDetector.kt
This file contains the `AudioBpmDetector` class, which is responsible for detecting the BPM of audio recorded from the device's microphone. It uses the Fast Fourier Transform (FFT) to analyze the audio data.

## app/src/main/java/com/hereliesaz/mademedance/MainActivity.kt
This is the main activity of the application. It's the entry point of the app and is responsible for managing the UI, handling sensor events (from the gyroscope), processing audio, and checking for a match between the movement BPM and the audio BPM.

## app/src/main/java/com/hereliesaz/mademedance/RhythmDetector.kt
This file contains the `RhythmDetector` class, which is responsible for detecting the user's movement BPM using the gyroscope sensor. It also uses FFT to analyze the sensor data.

## app/src/main/java/com/hereliesaz/mademedance/ui/ClipListScreen.kt
This file contains a Jetpack Compose screen that displays a list of recorded audio clips. The user can interact with the clips to play them or to try to identify the song.

## app/src/main/java/com/hereliesaz/mademedance/ui/MainScreen.kt
This file contains the main screen of the application, built with Jetpack Compose. It displays the movement BPM, the audio BPM, and the system status. It also has buttons to enable the microphone and to view the list of recorded clips.

## app/src/main/java/com/hereliesaz/mademedance/ui/theme/Color.kt
This file defines the color palette for the application's theme.

## app/src/main/java/com/hereliesaz/mademedance/ui/theme/Theme.kt
